# 抓包工具

## Fiddler 

安装fiddler 

[fiddler下载](https://www.telerik.com/fiddler)

安装完成后打开

```
# 界面 左边

# #请求列表
result 请求状态码
Protocal 请求http
host 端口    # 如果Tumel to 就是请求没有拿到
url  请求的地址   
```



fiddler 工具栏 Tools> Options

```
安装 Https的工具

```



访问网页遇到

```
devtool 禁用
	主网页这么弄 直接删除devtool 然后替换文件
	子网页这么弄 对于 devtool 返回一个空的请求,文件
```
